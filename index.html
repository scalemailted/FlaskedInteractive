Flasked Interactive

HTML + CSS first • light JS for glue & state

A minimalist interactive story set at the edge of a black hole. Each scene is a plain HTML page styled as a retro control panel. Prepare, explore, and decide whether to confront the anomaly—or escape.

▶ Play View Code
How to play:
  • Click on-screen buttons to make choices and move between scenes.
  • Watch HP and AP in the inventory panel; they’re passed via the URL between pages.
  • Use Travel to rest or load weapons before approaching hazards.
Built with just HTML, CSS, and tiny bits of JavaScript. No frameworks.
